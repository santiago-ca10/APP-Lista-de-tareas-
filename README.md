  **Descripción**

Aplicación de lista de tareas en Java usando Swing y el patrón MVC.
Permite agregar, editar, eliminar y completar tareas.

  **Tecnologías**

-Java 
-Swing (Interfaz Gráfica)
-Patrón MVC
j
  **Funcionalidades**

-➕ Agregar tareas
-🛠️ Editar tareas
-❌ Eliminar tareas
-✅ Marcar tareas como completadas
